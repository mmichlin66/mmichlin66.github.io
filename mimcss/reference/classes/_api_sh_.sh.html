Class sh

  • alpha(c: number | keyof INamedColors, a: number | string): IColorProxy

    Converts the given color and an optional alpha mask to the CSS Color representation. This method should be used when defining CSS color values in styleset properties. The color can be specified as a numeric value or as a string color name.

    • Defined in api/sh.ts:216

    Returns an object that can be assigned to the animation property.

    Parameters

    • name: Extended<StyleTypes.SingleAnimationName>

      Animation name. This can be either a string or a reference to the animation rule definition.

    • Default value duration: Extended<CssNumber> = 1000

      Animation duration. Integer numbers for milliseconds, floating point numbers for seconds. The default value is 1 second.

    • Default value func: Extended<StyleTypes.SingleAnimationTimingFunction> = "ease"

      Timing function. Default value is "ease". This can be one of the following types:

      • one of pre-defined timing function names.
      • a number of steps in the steps function. The step position will be set to jump-start.
      • one- or two-item array that defines a step function. The first item defines the number of steps. The optional second item is one of pre-defined step postion names.
      • four-item array that defines cubic-bezier function.
    • Default value delay: Extended<CssNumber> = 0

      Delay before the animation starts. Integer numbers for milliseconds, floating point numbers for seconds. The default value is 0.

    • Default value count: Extended<StyleTypes.SingleAnimationIterationCount> = 1

      Number of iterations the animation shold run. The default value is 1.

    • Default value direction: Extended<StyleTypes.SingleAnimationDirection> = "normal"

      Animation direction. The default value is "normal".

    • Default value mode: Extended<StyleTypes.SingleAnimationFillMode> = "none"

      Animation fill mode. The default value is "none".

    • Default value state: Extended<StyleTypes.SingleAnimationPlayState> = "running"

      Animation state. The default value is "running".

    Returns StyleTypes.SingleAnimation

Static rgb

  • rgb(r: number | string, g: number | string, b: number | string, a?: number | string): IColorProxy
    • Defined in api/sh.ts:85

    Converts the color specified as red, green, blue separation values and an optional alpha mask to a CSS color representation. This method should be used when defining CSS color values in styleset properties. Each color separation can be represented as a number or a string with the following meaning:

    • Integer number 0 to 255.
    • Floating number 0.0 to 1.0 non-inclusive, which is treated as percentage.
    • String which is used as is.

    The alpha mask can be one of the following:

    • Number 0 to 1 inclusive, which is treated as percentage.
    • String which is used as is.

    Parameters

    • r: number | string

      Red separation value.

    • g: number | string

      Green separation vaue.

    • b: number | string

      Blue separation value.

    • Optional a: number | string

      Optional alpha mask as a percentage value.

    Returns IColorProxy

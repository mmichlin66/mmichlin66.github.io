Class tsh

The msh class contains static helper functions that are used whenever there is a need to produce CSS string value based on more complicated type(s). The majority of these functions return StringProxy object so that they can be used in styleset properties assignments, for example:

<div style={{ color: tsh.rgb( 255, 128, 64) }}

Static custom

  • custom<T>(varDef: ICustomVar<T>, varValue: T): ICustomVal<T>
    • Defined in helpers/tsh.ts:339

    Defines a custom CSS property as part of a Styleset. Use it as in the following example:

    let myStyles = $scope( class
{
    mainColor = $custom( "color", "black");
    div = $tag( "div", { $custom: [ tsh.custom( this.mainColor, "brown") ] })
});

    This is equivalent to the following CSS:

    :root { --mainColor: "black"; }
div { --mainColor: "brown"; }

    The tsh.custom method will produce a compilation error if an invalid type is used for the property value.

    Type parameters

    • T

    Parameters

    • varDef: ICustomVar<T>
    • varValue: T

    Returns ICustomVal<T>

Static rgb

  • rgb(r: number | string, g: number | string, b: number | string, a?: number | string): StringProxy
    • Defined in helpers/tsh.ts:72

    Converts the color specified as red, green, blue separation values and an optional alpha mask to a CSS color representation. This method should be used when defining CSS color values in styleset properties. Each color separation cna be represented as a number or a string with the following meaning:

    • Integer number 0 to 255.
    • Floating number 0.0 to 1.0 non-inclusive, which is treated as percentage.
    • String which is used as is.

    The alpha mask can be one of the following:

    • Number 0 to 1 inclusive, which is treated as percentage.
    • String which is used as is.

    Parameters

    • r: number | string

      Red separation value.

    • g: number | string

      Green separation vaue.

    • b: number | string

      Blue separation value.

    • Optional a: number | string

      Optional alpha mask as a percentage value.

    Returns StringProxy

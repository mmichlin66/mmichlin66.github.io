Enumeration SchedulerType

The SchedulerType enumeration provides values used to define how the calls to the activate and deactivate functions schedule the writing of style changes to the DOM.

Index

Enumeration members

  • AnimationFrame

Enumeration members

AnimationFrame

AnimationFrame: = 2

Calls to activate and deactivate functions are accumulated until the next animation frame and then executed alltogether.

Manual

Manual: = 3

Calls to activate and deactivate functions are accumulated until the call to the forceDOMUpdate function and then executed alltogether.

Sync

Sync: = 1

Synchronous activation - style definitions are written to the DOM upon the activate and deactivate calls.

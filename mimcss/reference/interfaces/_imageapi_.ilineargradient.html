Interface ILinearGradient

The ILinearGradient interface represents a function that produces either linear-gradient or repeating-linear-gradient CSS function. It can be directly assigned to a suitable style property (e.g. background-image). In addition it has the to method that can be called to specify the angle of the gradiant.

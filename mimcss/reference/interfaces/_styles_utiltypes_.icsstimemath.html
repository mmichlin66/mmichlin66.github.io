calc

  • calc(formula: string, ...params: Extended<NumberBase<"Time" | "Percent">>[]): INumberProxy<"Time" | "Percent">

  • Inherited from INumberMath.calc

    • Defined in styles/UtilTypes.ts:196

    Creates property value using the CSS calc() function. This function accepts a formular string and an arbitrary number of parameters. The formular string can contain placeholders that will be replaced by the parameters. Placeholders have the following format:

    {<index> [| <unit>]}

    The <index> token is a zero-based index in the parameter array. The optional <unit> token is a measurement unit (length, percent, angle, etc.) and is used if the corresponding parameter is a number.

    class MyStyles
{
    wallGap = $var( "width", 16);
    myClass = $class({ maxWidth: Len.calc("100% - 2*{0}", this.wallGap)})
}

Interface IParameterizedPseudoClass

The IParameterizedPseudoClass interface maps names of pseudo classes that require parameters to the types that are used to specify these parameters. When a parameterized pseudo class is used as a property in the CombinedStyleset object, the value should be of the type from this interface.

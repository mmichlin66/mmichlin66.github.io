Interface IParameterizedPseudoElement

The IParameterizedPseudoElement interface maps names of pseudo elements that require parameters to the types that can be used to specify these parameters. When a parameterized pseudo element is used as a property in the CombinedStyleset object, the value should be of the type from this interface.

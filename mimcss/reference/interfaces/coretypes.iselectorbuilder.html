Interface ISelectorBuilder

Provides means to build complex selectors from multiple selector items of all possible kinds including tags, classess, IDs, attributes, pseudo classes and pseudo elements combined with CSS combinators. This interface is returned from the sel function.

Methods

adj

  • adj(...items: CssSelector[]): ISelectorBuilder

  • Adds one or more selector items to the existing selector as consecutive adjacent siblings. All items are concatenated and attached to the existing selector using the "+" combinator. Calling this method without any arguments inserts the "+" combinator.

    Example:

    class MyStyles extends css.StyleDefinition
{
    c1 = css.$class({...})
    c2 = css.$class({...})

    // produces css: section + .c1 + .c2 {...}
    s1 = css.$style( css.sel("section").adj( this.c1, this.c2), {...})
}

    Parameters

    • Rest ...items: CssSelector[]

      List of selectors to be added

    Returns ISelectorBuilder

and

  • and(...items: CssSelector[]): ISelectorBuilder

  • Adds one or more selector items to immediately follow the existing selector and each other. All items are concatenated and attached to the existing selector without any combinator. Calling this method without any arguments doesn't have any effect.

    Example:

    class MyStyles extends css.StyleDefinition
{
    c1 = css.$class({...})
    c2 = css.$class({...})

    // produces css: section.c1.c2 {...}
    s1 = css.$style( css.sel("section").and( this.c1, this.c2), {...})
}

    Parameters

    • Rest ...items: CssSelector[]

      List of selectors to be added

    Returns ISelectorBuilder

attr

  • attr(name: string, val?: string | number | boolean, cf?: "i" | "s", ns?: string | INamespaceRule): ISelectorBuilder
  • attr(name: string, op: AttrComparisonOperation, val: string | number | boolean, cf?: "i" | "s", ns?: string | INamespaceRule): ISelectorBuilder

  • Adds an attribute selector to immediately follow the existing selector. This variant doesn't accept comparison operation and, therefore, performs regular equality comparison. It also allows omitting the value, in which case, only the presence of the attribute is checked.

    Example:

    class MyStyles extends css.StyleDefinition
{
    // produces css: div:[title] {...}
    s1 = css.$style( css.sel("div").attr( "title"), {...})

    // produces css: a:[href="https://www.example.com"] {...}
    s2 = css.$style( css.sel("a").attr( "href", "https://www.example.com"), {...})
}

    Parameters

    • name: string

      Attribute name.

    • Optional val: string | number | boolean

      Attribute value - if omitted, only attribute presence is checked.

    • Optional cf: "i" | "s"

      Flag indicating whether or not attribute comparison is case insensitive. Undefined value means the comparison is case sensitive.

    • Optional ns: string | INamespaceRule

      Attribute's namespace.

    Returns ISelectorBuilder

  • Adds an attribute selector to immediately follow the existing selector.

    Example:

    class MyStyles extends css.StyleDefinition
{
    // produces css: a:[href^="https://"] {...}
    s1 = css.$style( css.sel("a").attr( "href", "^=", "https://"), {...})
}

    Parameters

    • name: string

      Attribute name.

    • op: AttrComparisonOperation

      Attrbute comparison operation.

    • val: string | number | boolean

      Attribute value - if omitted, only attribute presence is checked.

    • Optional cf: "i" | "s"

      Flag indicating whether or not attribute comparison is case insensitive. Undefined value means the comparison is case sensitive.

    • Optional ns: string | INamespaceRule

      Attribute's namespace.

    Returns ISelectorBuilder

nthChild

  • nthChild(nthExpr: NthExpression): ISelectorBuilder
  • nthChild(a: number, b: number): ISelectorBuilder

  • Adds the ":nth-child()" pseudo class with the given parameters.

    Example:

    class MyStyles extends css.StyleDefinition
{
    // produces css: p:nth-child("odd") {...}
    s1 = css.$style( css.sel("p").nthChild("odd"), {...})

    // produces css: p:nth-child(3) {...}
    s2 = css.$style( css.sel("p").nthChild(3), {...})

    // produces css: p:nth-child(3n) {...}
    s3 = css.$style( css.sel("p").nthChild([3]), {...})

    // produces css: p:nth-child(3n+1) {...}
    s4 = css.$style( css.sel("p").nthChild([3,1]), {...})

    // produces css: p:nth-child(3n-1) {...}
    s5 = css.$style( css.sel("p").nthChild([3,-1]), {...})
}

    Parameters

    • nthExpr: NthExpression

      String, number or tuple providing the value for the "nth" expression

    Returns ISelectorBuilder

  • Adds the ":nth-child()" pseudo class with the given parameters.

    Example:

    class MyStyles extends css.StyleDefinition
{
    // produces css: p:nth-child(3n+1) {...}
    s1 = css.$style( css.sel("p").nthChild(3, 1), {...})

    // produces css: p:nth-child(3n-1) {...}
    s2 = css.$style( css.sel("p").nthChild(3, -1), {...})
}

    Parameters

    • a: number

      Number before the "n" in the "An+B" expression

    • b: number

      Number after the "+" in the "An+B" expression. Use negative value to create a "An-B" expression.

    Returns ISelectorBuilder

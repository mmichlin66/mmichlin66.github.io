Interface ILengthMath

The ILengthMath interface contains methods that implement CSS mathematic functions on the <length> CSS types. This interface is implemented by the Len object.

calc

  • calc(formulaParts: TemplateStringsArray, ...params: Extended<CssLength>[]): IGenericProxy<"%" | LengthUnits>

  • Inherited from INumericMath.calc

    Creates property value using the CSS calc() function. This method is a tag function and must be invoked with a template string without parentheses. Parameters in the template string are of the type Extended<T>; that is, they can be either of the generic type T, or a CSS custom variable or constant of type T.

    Parameters

    • formulaParts: TemplateStringsArray
    • Rest ...params: Extended<CssLength>[]

      Array of parameters from the template string.

    Returns IGenericProxy<"%" | LengthUnits>

    Function implementing the IGenericProxy<U> callable interface. This allows the result of the calc method to be assigned only to the properties of a compatible numeric type.

    Example:

    class MyStyles extends css.StyleDefinition
{
    defaultPadding = css.var( "CssLength", 8)

    // calc( 200px - (2 * var(--defaultPadding)))
    cls1 = this.$class({
        width: css.Len.calc` 200px - (2 * ${this.defaultPadding})`
    })
}

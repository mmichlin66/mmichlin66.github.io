Interface IPercentMath

The IPercentMath interface contains methods that implement CSS mathematic functions on the <percent> CSS types. This interface is implemented by the Percent object.

calc

  • calc(formulaParts: TemplateStringsArray, ...params: Extended<CssPercent>[]): IGenericProxy<"%">

  • Inherited from INumericMath.calc

    Creates property value using the CSS calc() function. This method is a tag function and must be invoked with a template string without parentheses. Parameters in the template string are of the type Extended<T>; that is, they can be either of the generic type T, or a CSS custom variable or constant of type T.

    Parameters

    • formulaParts: TemplateStringsArray
    • Rest ...params: Extended<CssPercent>[]

      Array of parameters from the template string.

    Returns IGenericProxy<"%">

    Function implementing the IGenericProxy<U> callable interface. This allows the result of the calc method to be assigned only to the properties of a compatible numeric type.

    Example:

    class MyStyles extends css.StyleDefinition
    {
        defaultPadding = css.var( "CssLength", 8)

        // calc( 200px - (2 * var(--defaultPadding)))
        cls1 = this.$class({
            width: css.Len.calc` 200px - (2 * ${this.defaultPadding})`
        })
    }

clamp

  • clamp(min: Extended<CssPercent>, pref: Extended<CssPercent>, max: Extended<CssPercent>): IGenericProxy<"%">

  • Inherited from INumericMath.clamp

    Creates property value using the CSS clamp() function. Parameters are of the type Extended<T>; that is, they can be either of the generic type T, or a CSS custom variable or constant of type T.

    Parameters

    • min: Extended<CssPercent>

      Lower bound for the return value.

    • pref: Extended<CssPercent>

      Preferred value.

    • max: Extended<CssPercent>

      Upper bound for the return value.

    Returns IGenericProxy<"%">

    Function implementing the IGenericProxy<U> callable interface. This allows the result of the clamp method to be assigned only to the properties of a compatible numeric type.

    Example:

    class MyStyles extends css.StyleDefinition
    {
        // clamp( 200px, 25.5em, 45%)
        cls1 = this.$class({
            width: css.Len.clamp( 200, 25.5, css.percent(45))
        })
    }

Interface IClassNameRule

The IClassNameRule interface represents a combination of two or more class names. It can be used to make it easier to create elements with more than one CSS class. This interface is returned from the $classname function.

Objects implementing the IClassNameRule interface have the name property that contains the combined class name, e.g. "class1 class2". The cssClassName property contains the combined selector, e.g. "".class1.class2".

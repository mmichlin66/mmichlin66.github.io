Interface IGridLineRule

The IGridLineRule interface represents a definition of a named grid line. Objects implementing this interface are returned from the $gridline function or created when a grid area is defined using the $gridarea function.

Properties

Optional Readonly areaName

areaName?: string

Name of the grid area of which the line is either a start or an end line. It is defined only if the line name ends with "-start" or "-end".

Optional Readonly isStartEndOrNone

isStartEndOrNone?: boolean

Flag indicating whether the line is a start or end line of a grid area. The value is true if this is the start line; false if this is the end line; and undefined if the line is not related to any area.

Readonly name

name: string

Inherited from INamedEntity.name

Rule's name - this is a unique name that is assigned by the Mimcss infrastructure. This name doesn't have the prefix that is used when referring to classes (.), IDs (#) and custom CSS properties (--).

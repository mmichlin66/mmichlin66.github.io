Interface INamedEntity

The INamedEntity interface is a base interface implemented by all rules that have a name; that is, classes, IDs, keyframes, custom CSS properties, counters, grid lines and areas.

Readonly name

name: string

Rule's name - this is a unique name that is assigned by the Mimcss infrastructure. This name doesn't have the prefix that is used when referring to classes (.), IDs (#) and custom CSS properties (--).

Interface IPageRule

The IPageRule interface represents the CSS @page rule. Objects implementing this interface are returned from the $page function.

dependentRules: DependentRules

Inherited from IStyleRule.dependentRules

Object containing dependent rules. Property names are taken from special properties of the CombinedStyleset. This object allows callers to access dependent rules to change style property values programmatically.

  • Inherited from IStyleRule.setCustomProp

    Adds/replaces/removes the value of the given custmom CSS property in this rule.

    Type parameters

    • K: keyof IVarTemplateStyleset

    Parameters

    • customVar: IVarRule<K>

      IVarRule object defining a custom CSS property.

    • value: ExtendedVarValue<K>

      New value of the custom CSS property. If this value is undefined or null, the property is removed from the rule's styleset.

    • Optional important: boolean

      Flag indicating whether to set the "!important" flag on the property value.

    • Optional schedulerType: number

      ID of a registered scheduler type that is used to write the property value to the DOM. If undefined, the current default scheduler will be used.

    Returns void

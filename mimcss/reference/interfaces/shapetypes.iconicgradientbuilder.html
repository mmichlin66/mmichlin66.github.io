Interface IConicGradientBuilder

Represents an object that produces either conic-gradient or repeating-conic-gradient CSS function. It can be directly assigned to a suitable style property (e.g. background-image). In addition it has the from and at methods that can be called to specify the starting angle and center of the gradient.

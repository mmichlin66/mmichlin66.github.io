Interface ICrossFadeBuilder

Represents an invocation of the cross-fade() CSS function. It can be directly assigned to a suitable style property (e.g. background-image). Objects implementing this interface can be used wherever the CSS <image> type is used. In addition, it has convenient methods that can gradually build the parameters of the cross-fade() function.

Interface IRadialGradientBuilder

Represents an object that produces either radial-gradient or repeating-radial-gradient CSS function. It can be directly assigned to a suitable style property (e.g. background-image). In addition it has the circle, ellipse, extent and at methods that can be called to specify parameters of the gradient.

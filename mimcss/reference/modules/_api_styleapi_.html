    • Defined in api/StyleAPI.ts:164

    Returns an object that can be assigned to the animation property.

    Parameters

    • name: Extended<AnimationName_Single>

      Animation name. This can be either a string or a reference to the animation rule definition.

    • Default value duration: Extended<CssTime> = 1000

      Animation duration. Integer numbers for milliseconds, floating point numbers for seconds. The default value is 1 second.

    • Default value func: Extended<TimingFunction_Single> = "ease"

      Timing function. Default value is "ease". This can be one of the following types:

      • one of pre-defined timing function names.
      • a number of steps in the steps function. The step position will be set to jump-start.
      • one- or two-item array that defines a step function. The first item defines the number of steps. The optional second item is one of pre-defined step postion names.
      • four-item array that defines cubic-bezier function.
    • Default value delay: Extended<CssTime> = 0

      Delay before the animation starts. Integer numbers for milliseconds, floating point numbers for seconds. The default value is 0.

    • Default value count: Extended<AnimationIterationCount_Single> = 1

      Number of iterations the animation shold run. The default value is 1.

    • Default value direction: Extended<AnimationDirection_Single> = "normal"

      Animation direction. The default value is "normal".

    • Default value mode: Extended<AnimationFillMode_Single> = "none"

      Animation fill mode. The default value is "none".

    • Default value state: Extended<AnimationPlayState_Single> = "running"

      Animation state. The default value is "running".

    Returns Animation_Single

raw

  • raw(parts: TemplateStringsArray, ...params: any[]): StringProxy
    • Defined in api/UtilAPI.ts:97

    Returns a StringProxy function encapsulating the given string-like parameter. This function allows specifying arbitrary text for properties whose type normally doesn't allow strings. This is used as an "escape hatch" when a string value already exists and there is no sense to convert it to a proper type. This function is a tag function and must be invoked with the template string without parentheses.

The ExtendedStyleset type extends the Styleset type with certain properties that provide additional meaning to the styleset:

  • The $extends property specifies one or more parent style rules. This allows specifying parent rules using a convenient style-property-like notation. Parents can also be specified without a styleset.
  • The $important property specifies one or more names of styleset properties that shuld be considered "important". When the rule is inserted into DOM, the "!important" attribute is added to the property value.

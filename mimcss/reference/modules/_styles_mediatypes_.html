SingleMediaQuery

SingleMediaQuery: MediaFeatureset & { $mediaType?: MediaType; $negate?: undefined | false | true; $only?: undefined | false | true }
  • Defined in styles/MediaTypes.ts:77

Type representing a single set of styles as part of the @media rules. The styles in the styleset are combined with the "and" operator. The entire styleset can be negated, which will result in placing the "not" operator that will act on all styles in the query.

    • Defined in styles/UtilFuncs.ts:175

    Converts the given object to a CSS string.

    Parameters

    • val: any

      Object to convert to string.

    • propsAndFuncs: (string | [string, (val: any) => string, undefined | string])[]

      Array of property names and optionally functions. The order of the names determines in which order the properties should be added to the string. If a function is present for the property, it will be used to convert the property's value to the string. If a function is not present, then the property value should be converted to the string using the valueToString function.

    • Default value separator: string = " "
    • Default value usePropNames: boolean = false

      Flag indicating whether the names of the object's proprties should be added to the string.

SingleAngle_StyleType

SingleAngle_StyleType: number | Base_StyleType
  • Defined in styles/utils.ts:314

Type for CSS angle. Length can be represented using the following types:

  • string (e.g. 20deg or 1.4rad)
  • number: zero is treated as not having any suffix; integer numbers are treated as degrees; floating numbers are treated as radians.

SingleLength_StyleType

SingleLength_StyleType: "auto" | number | Base_StyleType
  • Defined in styles/utils.ts:227

Type for CSS length or percentage. Length can be represented using the following types:

  • string (e.g. 20px or 75%)
  • number: zero is treated as not having any suffix; integer numbers are treated as pixels; floating numbers are treated as percents: 0.0 to 1.0.

SingleTime_StyleType

SingleTime_StyleType: number | Base_StyleType
  • Defined in styles/utils.ts:384

Type for CSS time. Time can be represented using the following types:

  • string (e.g. 2s or 250ms)
  • number: integer numbers are treated as milliseconds while floating numbers are treated as seconds.

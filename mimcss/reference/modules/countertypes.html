ExtendedCounterStyleset

ExtendedCounterStyleset: {[ K in keyof ICounterStyleset]: CounterExtended<ICounterStyleset[K]> }

The ExtendedCounterStyleset type maps all @counter-style properties defined in the ICounterStyleset interface to the "extended" versions of their types. These extended types are defined using the CounterExtended generic type, which adds IRawProxy to the type that is defined in the IBaseCounterStyleset interface.

IFontFace

IFontFace: {[ K in keyof ICssFontFace]: FontFaceExtended<ICssFontFace[K]> }

The IFontFace type maps all @font-face properties defined in the ICssFontFace interface to the "extended" versions of their types. These extended types are defined using the FontFaceExtended generic type, which adds [[StringProxy]] and ICustomVar to the type that is defined in the ICssFontFace interface.

Type aliases

ExtendedFontFace

ExtendedFontFace: { [ K in keyof IFontFace]: FontFaceExtended<IFontFace[K]> }

The ExtendedFontFace type maps all @font-face properties defined in the IFontFace interface to the "extended" versions of their types. These extended types are defined using the FontFaceExtended generic type, which adds IRawProxy to the type that is defined in the IFontFace interface.

FontFaceExtended

FontFaceExtended<T>: T | IRawProxy

Type that extends the given type with the IRawProxy interface that allows specifying raw string value.

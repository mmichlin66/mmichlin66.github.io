Variables

Angle

Angle: IAngleMath = AngleMath

The Angle object contains methods that implement CSS mathematic functions on the <angle> | <percentage> CSS type. It implements the INumericMath interface and thus allows using the methods such as min, max, calc and clamp with parameters of the CssAngle type.

Frequency

Frequency: IFrequencyMath = FrequencyMath

The Frequency object contains methods that implement CSS mathematic functions on the <frequency> CSS type. It implements the INumericMath interface and thus allows using the methods such as min, max, calc and clamp with parameters of the CssFrequency type.

Len

Len: ILengthMath = LengthMath

The Len object contains methods that implement CSS mathematic functions on the <length> | <percentage> CSS type. It implements the INumericMath interface and thus allows using the methods such as min, max, calc and clamp with parameters of the CssLength type.

Example:

class MyStyles extends css.StyleDefinition
{
    defaultPadding = css.var( "CssLength", 8)

    // calc( 200px - (2 * var(--defaultPadding)))
    cls1 = this.$class({
        width: css.Len.calc` 200px - (2 * ${this.defaultPadding})`
    })
}

Num

Num: INumberMath = NumberMath

The Num object contains methods that implement CSS mathematic functions on the <number> CSS type. It implements the INumericMath interface and thus allows using the methods such as min, max, calc and clamp with parameters of the CssNumber type.

Example:

class MyStyles extends css.StyleDefinition
{
    // define custom CSS property - note that its value can be changed
    // programmatically
    columnQty = this.$var( "CssNumber", 3)

    // max( 5, var(--columnQty))
    cls1 = this.$class({
        columns: css.Num.max( 5, this.columnQty)
    })
}

Percent

Percent: IPercentMath = PercentMath

The Percent object contains methods that implement CSS mathematic functions on the <percentage> CSS type. It implements the INumericMath interface and thus allows using the methods such as min, max, calc and clamp with parameters of the CssPercent type.

Example:

class MyStyles extends css.StyleDefinition
{
    initialZoom = css.var( "CssPercent", 1.5)

    // min( 0.5, var(--initialZoom))
    cls1 = this.$class({
        zoom: css.Percent.min( 0.5, this.initialZoom)
    })
}
